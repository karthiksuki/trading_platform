# Code of Conduct

## Our Pledge

We want this project to be welcoming, respectful, and useful for everyone.

## Expected Behavior

- Be kind and constructive.
- Assume good intent.
- Give clear feedback without personal attacks.
- Respect different experience levels and backgrounds.
- Keep discussions focused on improving the project.

## Unacceptable Behavior

- Harassment, insults, threats, or discrimination.
- Sexualized language or attention.
- Publishing private information without permission.
- Trolling or repeatedly derailing discussions.

## Enforcement

Maintainers may remove comments, close issues, reject pull requests, or block contributors who violate this code of conduct.

## Reporting

If you see a problem, open a private maintainer contact channel when available. Include links, screenshots, and context so maintainers can respond fairly.
